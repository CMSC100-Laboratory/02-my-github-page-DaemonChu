Steps on how to create a GitHub page:
	1. Create a new repository
	2. Go to Settings tab and click Pages
	3. Under Build and Deployment:
		- Select "Deploy from a branch" as your Source
		- Select "main" as your Branch
		- click "Save"
	4. Github Page is now successfully published, and the live Website link can now be copied
